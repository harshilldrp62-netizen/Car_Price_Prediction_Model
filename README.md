# 🚗 Car Price Prediction using Multiple Linear Regression

## 📌 Overview

This project predicts the selling price of a used car using Machine Learning.
The model is built using Multiple Linear Regression and deployed as an interactive Streamlit dashboard.

Users can enter the car details such as:

- Car Name
- Company
- Manufacturing Year
- Kilometers Driven
- Fuel Type

and instantly receive the predicted market price.

## ✨ Features

✔ Interactive Streamlit Dashboard

✔ Real-time Car Price Prediction

✔ Data Cleaning & Preprocessing

✔ Multiple Linear Regression Model

✔ User-friendly Interface

✔ Model Serialization using Pickle

## 🎯 Problem Statement

Determining the fair market value of a used car is often difficult due to various influencing factors.

This project aims to develop a Machine Learning model capable of predicting the selling price of a used car based on its specifications.

Dataset Size : 819 records

Features:

1. Name
2. Company
3. Year
4. Kilometers Driven
5. Fuel Type

Target : Price

| Metric   | Value |
| -------- | ----- |
| R² Score | 0.79  |
| MAE      | 152030.65  |
| RMSE     | 290683.71  |
